# 数据类型

Go语言的数值类型包含不同大小的整数型、浮点数和负数，每种数值类型都有大小范围以及正负符号。

## 整数类型

| 类型 | 有无符号 | 占用存储空间 | 范围 | 备注 |
| :--- | :------- | :----------- | :--- | :--- |
| int  | 有       |              |      |      |
| uint | 无       |              |      |      |
| rune | 有       |              |      |      |
| byte | 无       |              |      |      |
